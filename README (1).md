# 📦 Remito App

Generador de remitos de venta, funciona como app en el celular (PWA).

## Archivos incluidos
- `index.html` — Estructura principal
- `style.css` — Diseño visual
- `app.js` — Lógica (filas, cálculo, guardado)
- `sw.js` — Service Worker (offline)
- `manifest.json` — Configuración PWA

## Cómo usarla

### Opción 1 — GitHub Pages (recomendado)
1. Creá un repositorio en GitHub y subí estos 5 archivos
2. Activá GitHub Pages (Settings → Pages → main branch)
3. Entrá al link desde el celular e instalá la app

### Opción 2 — Local
Abrí `index.html` directo en el navegador (sin service worker activo, funciona igual)

## Instalación en el celu
- **Android (Chrome):** Menú ⋮ → "Instalar aplicación"
- **iOS (Safari):** Compartir ↑ → "Agregar al inicio"

## Características
- ✅ Pantalla de inicio tipo app
- ✅ Guardado de datos de empresa (queda en el celu)
- ✅ Campos en blanco para que cualquiera lo personalice
- ✅ Suma automática de totales
- ✅ Agregar / eliminar filas
- ✅ Impresión / PDF
- ✅ Funciona sin internet después de la primera carga
